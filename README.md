# -PyTorch- 深度学习与PyTorch入门实战教程 

-- 简介
基本的数学理论，线性回归，逻辑回归，梯度及梯度下降，损失函数，多分类问题，BatchNorm，卷积神经网络CNN/ResNet，
循环神经网络RNN/LSTM，对抗生成网络GAN/WGAN等等，以及对应的PyTorch的实现方式讲解。通过理论与实践相结合的教学方式，
学员可以最大程度理解并掌握算法。
